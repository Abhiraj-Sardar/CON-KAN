📖 What is a CNN?

A Convolutional Neural Network is a specialised deep learning architecture designed to process data with a known grid-like topology — most famously 2-D images. CNNs exploit three key ideas:

  • 🔲 Local connectivity — each neuron sees only a small receptive field
  • 🔁 Weight sharing — the same filter scans the entire input
  • 📐 Translation invariance — pooling makes features position-agnostic

⚙️ How Convolution Works

A filter (kernel) of size k × k is slid across the input with stride s, computing an element-wise dot product at every position to produce an activation / feature map. For an input of size W × H × C and F filters:

Output W = (W − k + 2p) / s + 1

Multiple stacked convolutional layers build a hierarchical feature extractor — from edges and blobs in early layers to complex object parts in deeper layers.
