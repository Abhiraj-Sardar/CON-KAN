📖 What is KAN?

KAN (Kolmogorov–Arnold Network) is a novel neural architecture grounded in the Kolmogorov–Arnold representation theorem, which proves that any continuous multivariate function $ f(x_1, \ldots, x_n) $ can be decomposed as:

f(x₁,…,xₙ) = Σ Φq( Σ φq,p(xp) )

KAN directly realises this decomposition as a trainable network, placing learnable univariate functions on each edge rather than fixed activations at nodes.

⚙️ Architecture Details

Each KAN layer consists of a grid of B-spline basis functions per edge, whose control points are learned during back-propagation. A KAN with width vector [n₀, n₁, …, nₗ] stacks l such layers, each mapping its input through ni × ni+1 trainable univariate splines.

✅ Strengths vs MLP

  • 🔍 Interpretable — activations can be visualised and simplified
  • 📐 Accurate — matches or beats MLP with fewer parameters
  • 🧮 Symbolic — can distil learned functions into closed-form expressions
  • 🌱 Continual learning — localised splines reduce catastrophic forgetting

⚠️ Current Limitations

  • 🐢 Slower training than MLPs of comparable size
  • 🖼️ Raw image tasks need CNN hybrid front-ends
  • 📚 Tooling & community still maturing
